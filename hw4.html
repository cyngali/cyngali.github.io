Lists Usage
Unordered ListUsed to present a list of items in no specific order. To implement this list you use the tag <ul>
Ordered ListUsed to present a list of items in a specific order. To implement this list you use the tag <ol>
Definition ListUsed to present a term to be defined followed by a definition. To implement this list you use the tag <dl>
HTML
Stands for HyperText Markup Language, it is the basic building block of a webpage and is used for creating and visually representing a webpage.
CSS
Stands for Cascading Style Sheets, it is used to define styles for a web page including design, layout, and variations in display.
FTP
Stands for File Transfer Protocol, it is the process that involves the transfer of files between devices over a network.
SSH
Also known as Secure Shell is a method for secure remote login from one computer to another.
